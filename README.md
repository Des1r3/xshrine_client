# xshrine_client

xshrine：xray后端客户端

### 准备

1. 安装 python3 环境并更新 pip 版本：
```bash
yum install python3
pip3 install --upgrade pip
```

2. 安装 python3-devel 和 gcc
```bash
yum install gcc-c++ python3-devel
```

### 安装

1. 使用脚本安装xray：
```bash
sh install-xray.sh
```

2. 使用脚本安装xshrine客户端（需要修改参数）：
```bash
sh install-xshrine.sh
```
